# mem-agent-mcp

## Running Instructions

1. `make check-uv`
2. `make install`
3. `make setup`
4. `make run-agent`
5. `make generate-mcp-json`
6. Instructions per app/provider:
    - Claude Desktop:
        - Copy the generated `mcp.json` to the wherever your `claude_desktop.json` is located, then, quit and restart Claude Desktop.
